My 8-bit breadboard CPU is a project inspired by Ben Eater's [eponymous series](https://youtube.com/playlist?list=PLowKtXNTBypGqImE405J2565dvjafglHU&si=hM_gdIcFqrUBWIJ8) and heavily influenced by GitHub user Fadil-1's [improvements to the original design.](https://fadil-1.github.io/blog/8-bit_breadboard_CPU/overview/) These designs, in turn, are based on the "SAP" (**S**imple **a**s **P**ossible) von Neumann architectures described in Albert Paul Malvino's *Digital Computer Electronics*.

## Motivation and focus of this project

My primary motivation for this project was to bridge the gap in my knowledge between basic electronics (*e.g.*, current, resistance, voltage, and so on) and abstract user-space (*e.g.*, a word processor or web browser). To distill this into one question:

> **Between pressing the power button and seeing a login screen, what is actually going on?**

Throughout this site, I will refer to Ben Eater's design as the **SAP-1** and Fadil-1's design as the **F8-BB**.
